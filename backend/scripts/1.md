# Summary

## Getting Started

### Definition

- Axios: promised-based HTTP Client for nodejs and browser.
- Use nodejs http module on server-side, XMLHttpRequest on client (browser).

### Features

- Make XMLHttpRequest, http request.
- Support Promise API.
- Intercept, transform req and res; cancle req, timeout.
- Auto req body serialization: JSON, Multipart/FormData, URL encoded form.
- Auto JSON Data handling.
- Extra info (speed rate, remaining time), set bandwidth limit.
- Protect XSRF.
